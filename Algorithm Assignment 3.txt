# PROBLEM NO.3
Ask the user to enter number 1.
Ask the user to enter number 2.
Ask the user to enter arithmetic operator(+,-,*,/,%).
If arithmetic operator == + Then
 Set ans to ( number 1 + number 2).
 Print ans.
Else If arithmetic operator == - Then
 Set ans to ( number 1 - number 2).
 Print ans.
Else If arithmetic operator == * Then
 Set ans to ( number 1 * number 2).
 Print ans.
Else If arithmetic operator == / Then
 Set ans to ( number 1 / number 2).
 Print ans.
Else If arithmetic operator == % Then
 Set ans to ( number 1 % number 2).
 Print ans.
Else
 Print 'Invalid Operator'.
